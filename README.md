# Piano-Hand-API
API using Music 21 and MIDI to convert midi files and piano notes to coordinates for a piano hand.

The KiCAD files have a full PCB design for the barrel jack, LM2896, motor servers, stepper motor driver, and microphone. This includes the Gerber files, drill files, and bill of materials. It also includes the libraries for the symbols and footprints used in this project.
